# Tiffin - Hostel Mess Management

Full-stack Next.js app for managing hostel mess attendance.

## Setup

1. Clone and `npm install`
2. Copy `.env.example` to `.env` and fill values
3. `npx prisma migrate dev`
4. `npx prisma db seed`
5. `npm run dev`

## Roles

- Super Admin (admin@tiffin.edu / admin123)
- Warden (warden.blockA@tiffin.edu / warden123)
- Student (usernames generated by wardens)

## Tech Stack

Next.js 16, React 19, Tailwind CSS 4, Prisma, PostgreSQL, NextAuth.js
